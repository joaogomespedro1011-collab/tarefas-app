# 📝 App de Tarefas - Next.js

Aplicação simples de gerenciamento de tarefas desenvolvida com Next.js 15, TypeScript e testes com Jest + Testing Library.

---

## 🚀 Tecnologias utilizadas

* Next.js 15 (App Router)
* React
* TypeScript
* Jest
* Testing Library

---

## 📦 Instalação

Acesse a pasta do projeto:

```bash
cd tarefas-app
```

Instale as dependências:

```bash
npm install
```

---

## ▶️ Executando o projeto

Para rodar a aplicação:

```bash
npm run dev
```

Abra no navegador:

```
http://localhost:3000
```

---

## 🧪 Executando os testes

Rodar testes:

```bash
npm test
```

Modo watch:

```bash
npm test -- --watch
```

Cobertura de testes:

```bash
npm test -- --coverage
```

---

## 📌 Funcionalidades

* Listagem de tarefas (simulação de API)
* Adição de novas tarefas
* Contador de tarefas com hook customizado
* Testes unitários com Jest
